# ITS Voice RAG Bot — unified requirements
#
# ⚠️  PYTHON VERSION: 3.10, 3.11, or 3.12 ONLY. Verified on 3.11.
#     Python 3.13 and 3.14 WILL FAIL because NVIDIA's moshi-personaplex
#     fork pins `torch<2.5`, and torch only ships 3.13+ wheels from 2.5
#     onward (and 3.14 wheels from 2.9). There is no overlap. This is not
#     fixable by relaxing pins — the PersonaPlex model literally requires
#     the older torch.
#
# The vendored moshi-personaplex fork (./personaplex/moshi) is the tightest
# constraint in this project; its pyproject.toml pins numpy, sphn,
# huggingface-hub, einops, sentencepiece, torch, and aiohttp. Every other
# dep below is chosen to be compatible with those pins.
#
# Install (from a clean slate):
#   python3.11 -m venv .venv311
#   source .venv311/bin/activate
#   pip install -r requirements.txt
#
# If you already have newer sphn / huggingface-hub in the venv, pip will
# downgrade them — that is expected and required for PersonaPlex to load.

#   sphn>=0.1.4,<0.2         torch>=2.2.0,<2.5
